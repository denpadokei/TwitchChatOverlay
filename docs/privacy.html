TwitchChatOverlay Privacy Policy

このプライバシーポリシーは、TwitchChatOverlay が Twitch および YouTube API Services を利用する際に、どの情報へアクセスし、どのように保存、利用、削除するかを説明するものです。YouTube 機能を利用する前に必ず確認してください。

1. アプリの概要

TwitchChatOverlay は、Twitch と YouTube のライブ配信イベントやライブチャットを Windows デスクトップ上の通知として表示するためのデスクトップアプリです。YouTube 機能は YouTube API Services を利用しています。

2. 取得または保存する情報

このアプリは、YouTube 機能で次の情報を扱います。

  • YouTube OAuth アクセストークン
  • YouTube OAuth リフレッシュトークン
  • トークン取得日時を含む認可状態表示文字列
  • YouTube ライブチャットから受信した表示名、メッセージ本文、Super Chat 情報、メンバーシップ情報

ライブチャットの表示データは通知表示のためにメモリ上で処理されます。トークン以外の YouTube ユーザーデータやメッセージ履歴を永続保存する機能は想定していません。

3. 利用目的

  • YouTube ライブチャット、Super Chat、メンバーシップ通知を表示するため
  • 配信開始検出とライブチャット接続を維持するため
  • ユーザーが再認可せずにアプリを再利用できるようにするため

このアプリは、取得した YouTube データを広告配信、販売、第三者への再提供、プロフィール分析、派生指標の販売のためには使用しません。

4. 保存場所と保護方法

YouTube OAuth トークンおよびアプリ設定は、Windows のユーザー単位保護機能を利用して暗号化したうえで、ローカル環境の設定ファイルに保存されます。保存先は通常、ユーザーの Application Data 配下の TwitchChatOverlay 設定ファイルです。

このアプリは、YouTube OAuth で読み取り専用スコープのみを要求します。

5. 第三者共有

このアプリが取得した YouTube OAuth トークンや設定情報を、アプリの開発者が外部サーバーへ送信したり、第三者へ販売したりすることは意図していません。YouTube API Services への通信、およびユーザーが明示的に開いた Google または YouTube のページへのアクセスは、それぞれ Google または YouTube のサービスとして処理されます。

6. 同意の取り消しと保存データの削除

ユーザーは次の方法で保存済みデータや権限を管理できます。

  • アプリ内の YouTube タブからローカル保存済み YouTube 認可情報を削除する
  • アプリ内の YouTube タブから Google 側の権限取り消しを試行する
  • Google の権限管理ページからこのアプリへのアクセスを手動で取り消す

Google 側で権限を取り消した後も、ローカル保存済みデータを削除したい場合はアプリ内の削除操作も実行してください。ローカル削除は YouTube 側に保存されているデータを削除するものではありません。
