Windows desktop app for stream notifications

TwitchChatOverlay

TwitchChatOverlay は、Twitch と YouTube のライブチャット、Super Chat、メンバーシップ通知などを Windows デスクトップ上に表示する配信者向けアプリです。YouTube 機能は YouTube API Services を利用し、読み取り専用権限で動作します。

Brand verification summary

  • App name: TwitchChatOverlay
  • Platform: Windows desktop application
  • YouTube scope: youtube.readonly only
  • Primary use: local on-screen notifications for broadcasters

アプリの役割

このアプリは配信中のコメントやイベントをローカル PC 上で見やすく表示するための補助ツールです。動画のアップロード、コメント投稿、プレイリスト編集、チャンネル管理などの書き込み操作は行いません。

Google に伝わるべき点

  • YouTube 機能は YouTube Live Chat の読み取りと通知表示のために使われます。
  • OAuth トークンはローカル端末にのみ保存されます。
  • ユーザーはアプリ内からローカル削除と Google 側の権限取り消しを実行できます。

English summary

TwitchChatOverlay is a Windows desktop app for streamers. It reads Twitch and YouTube live events and displays them locally as on-screen notifications. The YouTube integration uses YouTube API Services with read-only access.

This page exists to identify the app, link to its privacy policy and terms, and provide a public home page for OAuth branding verification.
