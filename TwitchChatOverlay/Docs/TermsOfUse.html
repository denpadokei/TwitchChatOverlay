TwitchChatOverlay 利用条件

TwitchChatOverlay の YouTube 機能を利用することで、ユーザーは YouTube の関連規約およびこのアプリの利用条件に同意したものとみなされます。

1. YouTube 利用規約への同意

このアプリの YouTube 機能を利用することで、ユーザーは YouTube 利用規約 に従うことに同意したものとみなされます。YouTube API Services の利用には、YouTube API Services Terms of Service と YouTube Developer Policies も適用されます。

2. アプリが提供する機能

このアプリの YouTube 機能は、YouTube Live Chat の読み取りと通知表示を目的としたものです。アップロード、コメント投稿、プレイリスト操作などの書き込み機能は提供しません。

3. 認可と権限

このアプリは YouTube 読み取り専用権限を要求します。ユーザーは Google の認可画面で権限内容を確認し、必要に応じて認可を拒否または後から取り消すことができます。

4. 保存済みデータの削除

アプリ内の YouTube タブから、ローカル保存済みの認可情報を削除できます。Google 側の権限も停止したい場合は、アプリ内の権限取り消し操作、または Google 権限管理ページ を利用してください。

5. 免責

このアプリは現状有姿で提供されます。YouTube、Google、Twitch の仕様変更、API 制限、認可設定、ネットワーク障害、または外部サービスの変更により、一部機能が利用できなくなる場合があります。

6. 問い合わせ先

このアプリの利用条件や運用に関する問い合わせは、GitHub Discussions を利用してください。
